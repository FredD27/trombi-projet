The Original Gangsters (OG)

Les OGs, ou Original Gangsters, sont les membres d'une équipe de choc dont la spécialité est... l'imprévisible. Que font-ils? Que pensent -ils? Personne n'en est vraiment sûr, peut-être même pas eux. Les dernières informations les concernant font mention d'une formation dans le développement web, mais ces cancres seront sûrement banni rapidement.
